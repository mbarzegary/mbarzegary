# Hey there! I'm Mojtaba (Moji) 👋

**Computational engineering researcher** specializing in modeling, simulation, and optimization of complex multiphysics systems.

I'm a **Marie Curie Postdoctoral Fellow** at [Eindhoven University of Technology](https://www.tue.nl/en/) (the Netherlands), working at the intersection of scientific computing, high-performance computing, and electrochemical engineering. Previously, I did my PhD in computational biomedical engineering at [KU Leuven](https://www.kuleuven.be/english/kuleuven/) (Belgium).

I'm passionate about developing open-source scientific software, scaling simulations on HPC clusters, and applying machine learning to engineering problems. Outside of work, you might find me occasionally playing bass guitar or studying chess.

### 🔬 Currently Working On

- **Topology optimization** of porous electrodes for redox flow batteries
- **GPU-accelerated** multiphysics simulations using CUDA, Firedrake, and OpenFOAM
- Hybrid **ML + physics-based** approaches for electrochemical system design
- Mentoring PhD & MSc students in computational modeling

### 🛠️ Featured Open-Source Projects

| Project | Description | Stack |
|---------|-------------|-------|
| [**RedOx-TO**](https://github.com/mbarzegary/RedOx-TO) | Topology optimization framework for porous electrodes in electrochemical flow reactors | Python, Firedrake, PETSc |
| [**RfbFoam**](https://github.com/mbarzegary/RfbFoam) | Redox flow battery simulation tool (fluid, mass & charge transport) | C++, OpenFOAM |
| [**BioDeg**](https://github.com/mbarzegary/BioDeg-UI) | Simulation of corrosion & biodegradation in metallic biomaterials | C++, FreeFEM, PETSc, Qt |
| [**ASLI**](https://github.com/tpms-lattice/ASLI) | TPMS lattice structure generator for engineering applications | C++, CGAL, Mmg |
| [**Skeletal Cell Atlas**](https://github.com/mbarzegary/skeletal-cell-atlas) | Interactive web-based genomics resource for skeletal developmental data | Python, Dash, Plotly |
